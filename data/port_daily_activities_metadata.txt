PORT DAILY ACTIVITIES - COLUMN METADATA
Dataset: Port Daily Activities
Description: Daily vessel port calls and cargo trade volume data by port,
             disaggregated by vessel/cargo type.

TEMPORAL IDENTIFIERS
Column       : date
Type         : Date (YYYY-MM-DD)
Description  : Full calendar date of the recorded activity.
Example      : 2023-01-15

Column       : year
Type         : Integer
Description  : Calendar year extracted from the date field.
Example      : 2023

Column       : month
Type         : Integer (1–12)
Description  : Calendar month extracted from the date field.
Example      : 1 (January)

Column       : day
Type         : Integer (1–31)
Description  : Calendar day extracted from the date field.
Example      : 15

PORT IDENTIFIERS
Column       : portid
Type         : String / Integer
Description  : Unique identifier for each port.
Example      : 10234

Column       : portname
Type         : String
Description  : Official name of the port.
Example      : Port of Mombasa

Column       : country
Type         : String
Description  : Full name of the country where the port is located.
Example      : Kenya

Column       : ISO3
Type         : String (3-character ISO 3166-1 alpha-3 code)
Description  : International standardized 3-letter country code.
Example      : KEN

PORT CALLS — BY VESSEL TYPE
(Number of vessel arrivals/departures recorded at the port on the given date)
Column       : portcalls_container
Type         : Integer
Description  : Number of port calls by container vessels.

Column       : portcalls_dry_bulk
Type         : Integer
Description  : Number of port calls by dry bulk carrier vessels.

Column       : portcalls_general_cargo
Type         : Integer
Description  : Number of port calls by general cargo vessels.

Column       : portcalls_roro
Type         : Integer
Description  : Number of port calls by Roll-on/Roll-off (RoRo) vessels.

Column       : portcalls_tanker
Type         : Integer
Description  : Number of port calls by tanker vessels (liquid cargo).

Column       : portcalls_cargo
Type         : Integer
Description  : Total port calls across all cargo vessel types
               (sum of container, dry bulk, general cargo, roro, tanker).

Column       : portcalls
Type         : Integer
Description  : Total port calls across all vessel types, including
               non-cargo vessels (e.g. passenger, service).

IMPORTS — BY CARGO TYPE
(Volume of goods imported through the port on the given date)
Column       : import_container
Type         : Numeric (tonnes or TEUs)
Description  : Import volume carried by container vessels.

Column       : import_dry_bulk
Type         : Numeric (tonnes)
Description  : Import volume of dry bulk commodities (e.g. grain, coal, ore).

Column       : import_general_cargo
Type         : Numeric (tonnes)
Description  : Import volume of general/break-bulk cargo.

Column       : import_roro
Type         : Numeric (tonnes or units)
Description  : Import volume of roll-on/roll-off cargo (e.g. vehicles).

Column       : import_tanker
Type         : Numeric (tonnes)
Description  : Import volume of liquid bulk cargo (e.g. oil, chemicals).

Column       : import_cargo
Type         : Numeric (tonnes)
Description  : Total import volume across all cargo types.

Column       : import
Type         : Numeric (tonnes)
Description  : Total import volume including all vessel and cargo types.

EXPORTS — BY CARGO TYPE
(Volume of goods exported through the port on the given date)
Column       : export_container
Type         : Numeric (tonnes or TEUs)
Description  : Export volume carried by container vessels.

Column       : export_dry_bulk
Type         : Numeric (tonnes)
Description  : Export volume of dry bulk commodities (e.g. grain, coal, ore).

Column       : export_general_cargo
Type         : Numeric (tonnes)
Description  : Export volume of general/break-bulk cargo.

Column       : export_roro
Type         : Numeric (tonnes or units)
Description  : Export volume of roll-on/roll-off cargo (e.g. vehicles).

Column       : export_tanker
Type         : Numeric (tonnes)
Description  : Export volume of liquid bulk cargo (e.g. oil, chemicals).

Column       : export_cargo
Type         : Numeric (tonnes)
Description  : Total export volume across all cargo types.

Column       : export
Type         : Numeric (tonnes)
Description  : Total export volume including all vessel and cargo types.


NOTES
- "cargo" aggregates = container + dry_bulk + general_cargo + roro + tanker.
- "portcalls" (total) may include vessel types not broken out individually.
- ISO3 codes follow ISO 3166-1 alpha-3 standard.
Total Columns: 29
